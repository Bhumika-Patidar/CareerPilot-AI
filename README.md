🚀 AI Resume Optimizer & Job Recommendation Platform (Frontend)

This repository contains the frontend prototype of my upcoming full-stack AI-powered career assistance platform.

✨ Features Implemented (Frontend UI)

📄 Resume Upload Section
- Upload resume files (PDF, DOCX, TXT, DOC)
- Option to paste resume text manually

📝 Job Description Input
- Paste a job description for analysis
- User-friendly text area for quick input

🤖 AI Generated Resume Section
- Displays the optimized resume generated by AI
- Download option for exporting the generated resume as PDF

🔍 Resume Analysis Button
- Triggers the resume optimization workflow
- Will be connected to AI processing in the backend

💼 Recommended Jobs Section
- Displays jobs matched according to the optimized resume
- Planned integration with LinkedIn, JSearch API, and other job sources

⚠️ Note:
This is currently only the frontend implementation of the project.
Backend services, AI resume optimization, ATS analysis, PDF generation, and job recommendation logic are under development and will be integrated in future updates.
